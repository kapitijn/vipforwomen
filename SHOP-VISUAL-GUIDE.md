## Mobile Responsive

All sections are **fully responsive**:

**Desktop (1024px+):**
- 4 products per row
- Large category cards
- Full countdown timer

**Tablet (768px - 1023px):**
- 2-3 products per row
- Medium category cards
- Compact countdown

**Mobile (<768px):**
- 1 product per column
- Stacked category cards
- Simplified countdown
- Touch-friendly buttons

## When Sections Show

**Main Shop Page (`/shop`):**
✅ Shows ALL sections (Hero + Flash + Categories + Best + Gifts + Products)

**Filtered Pages:**
❌ Hides featured sections, shows only filtered products:
- `/shop?category=dresses` - Only dresses
- `/shop?on_sale=true` - Only sale items
- `/shop?orderby=popularity` - Only popular items
- `/shop?search=bag` - Only search results

This keeps the page clean and focused when users are looking for specific items!
