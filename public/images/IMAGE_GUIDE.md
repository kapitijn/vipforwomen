# Image Upload Guide for VIP For Women

## Where to Upload Images

### 1. **Product Images** ✅ EASIEST
Upload through **WooCommerce Admin Panel**:
- Login to WordPress admin
- Go to Products → Add New or Edit Product
- Upload images in the Product Gallery section
- These automatically appear on your website!

### 2. **Homepage Banner Images** (Horizontal Scroll Section)
Upload to: `/public/images/banners/`

**Required files:**
- `summer-deals.jpg` (500x600px minimum)
- `new-in-store.jpg` (500x600px minimum)
- `new-bags.jpg` (500x600px minimum)
- `hot-deals.jpg` (500x600px minimum)

**Recommended specs:**
- Size: 500x600 pixels (or larger, same ratio)
- Format: JPG or PNG
- File size: Under 500KB each (compressed)
- Quality: High quality, but optimized for web

### 3. **Hero/Featured Images** (Future use)
Upload to: `/public/images/hero/`

---

## How to Upload Images

### **Method 1: Bluehost File Manager (Easiest)**
1. Login to your Bluehost account
2. Click on "Advanced" or "cPanel"
3. Find "File Manager"
4. Navigate to: `public_html/public/images/banners/`
5. Click "Upload" button
6. Select your images
7. Done! Images appear immediately

### **Method 2: FTP Client (FileZilla)**
1. Download FileZilla: https://filezilla-project.org/
2. Connect with FTP credentials (from Bluehost)
3. Navigate to: `/public_html/public/images/banners/`
4. Drag and drop images from your computer
5. Done!

### **Method 3: WordPress Media Library + Plugin**
Install "WP File Manager" plugin:
1. WordPress Admin → Plugins → Add New
2. Search "WP File Manager"
3. Install and activate
4. Now you can upload files through WordPress dashboard!

## Image Requirements

### Banner Images (Horizontal Scroll)
- **Dimensions:** 500x600px (portrait orientation)
- **Aspect Ratio:** 5:6 (keep this ratio if using larger sizes)
- **Format:** JPG (preferred) or PNG
- **Size:** 200-500KB per image
- **Style:** High fashion, luxury aesthetic, black/white/silver tones work best

### Product Images (WooCommerce)
- **Dimensions:** 800x800px or larger (square)
- **Format:** JPG or PNG
- **Size:** Under 300KB
- **Background:** White or transparent recommended

---

## Tips for Best Results

1. **Compress images before uploading:**
   - Use https://tinypng.com/ or https://squoosh.app/
   - Reduces file size without losing quality
   - Faster website loading!

2. **Use consistent dimensions:**
   - Keeps website looking professional
   - Prevents stretching or distortion

3. **Name files clearly:**
   - Use lowercase
   - Use hyphens instead of spaces
   - Example: `summer-sale-2024.jpg`

4. **Keep backups:**
   - Save original high-res files on your computer
   - You can always re-upload if needed
